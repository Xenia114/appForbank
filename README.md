# appForbank
Android-приложение для клиентов банка
Описание функциональности системы:
1. Предоставление клиентам круглосуточного доступа к счетам через интернет.
2. Возможность просмотра баланса, истории операций, выписок.
3. Проведение платежей (переводы, оплата услуг, кредиты).
4.Поддержка различных уровней авторизации (пароль, SMS, биометрия).
5. Высокий уровень защиты данных (SSL, двухфакторная аутентификация).
6. Наличие адаптивного интерфейса для смартфонов и планшетов.
7. Генерация отчетов по операциям.
8. Ведение журнала действий пользователя.
9. Интеграция с внешними системами (платежные сервисы, Центральный банк РФ).
